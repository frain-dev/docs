You can configure your license key with any of the following:
<CodeGroup>
  ```json config file
    {
      ...
      "license_key": "{insert license key}"
      ...
    }
  ```

  ```bash cli flag
    convoy server --license-key "{insert license key}"
  ```

  ```bash environment variable
    CONVOY_LICENSE_KEY="{insert license key}"
  ```
</CodeGroup>
